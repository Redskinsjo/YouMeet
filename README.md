## Getting Started

Need [node](https://nodejs.org/fr/) installed and yarn: ```npm i -g yarn```

Clone the repo with:
```git clone https://github.com/Redskinsjo/YouMeet.git```

Install dependencies with:
```yarn install```

Run the environment with:
```yarn run dev```

and the tests with:
```yarn run test```
